# pet-react-vite-ts-app

Приложение магазина с возможностью авторизации, поиска товаров и добавления их в корзину.

## Используемые технологии:
  * React, React-DOM, React Router, Redux Toolkit, TypeScript.
  * Vite для сборки приложения.
  * API от https://dummyjson.com/docs/products для получения данных о товарах.
  * API от https://dummyjson.com/docs/auth для авторизации.
   > [!IMPORTANT]
  > Данное API не предоставляет возможности регистрировать новых пользователей. По этой причине на странице авторизации и "регистрации" используются данные уже зарегистрированного пользователя.
  
  > [!NOTE]
  > Для надежной работы данного API рекомендуется использовать **virtual private network.**
  
## Особенности:
  ### Запросы на сервер
  * Реализация с помощью Axios.
  ### CSS
  * Использован пакет classnames для некоторых компонентов.
  ### Логика управления состоянием приложения
  * Реализация с помощью Redux Toolkit.
  
## Запуск
  * npm install
  * npm run dev
